# LibraryManagementSystem-FastAPI
A RESTful Library Management System built using FastAPI that allows users to manage books, issue and return them, and perform search and count operations. This project demonstrates core backend concepts including API design, data validation, and in-memory data handling.


This project is a simple backend application built using FastAPI that simulates a Library Management System. It allows users to perform operations like adding books, viewing all books, issuing and returning books, searching, and counting total books.

##  Features

-  Add new books
-  View all books
-  Search books by title or author
-  Count total books
-  Issue a book
-  Return a book
-  Handle errors (book not found, already issued, etc.)

##  Tech Stack

- Python
- FastAPI
- Uvicorn
- Pydantic (for validation)
